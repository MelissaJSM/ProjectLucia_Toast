# ProjectLucia_Toast
프로젝트 루시아의 기능 중 알람 기능 프로그램 입니다.


🛡️ 개인정보 처리 및 보안 안내 (Privacy & Security Guidelines)
Project Lucia는 사용자와 더욱 긴밀하게 소통하고 반응하는 인공지능 어시스턴트를 구현하기 위해 만들어졌습니다. 이를 위해 운영체제 및 메신저의 일부 데이터를 읽어오는 기능이 포함되어 있으므로, 사용 전 아래 사항을 반드시 숙지해 주시기 바랍니다.

1. 수집하는 데이터 및 목적

Windows 시스템 알림 (Toast Notification): 디스코드, 메일 등 시스템 알림 내용을 감지하여 AI가 상황에 맞는 대화를 선톡으로 건넬 수 있도록 합니다.

카카오톡 화면 캡처: 카카오톡 메시지 알림 시 화면의 일부를 캡처(이미지)하여, AI가 사용자의 대화 흐름을 시각적으로 인지하고 멀티모달 형태로 반응하도록 돕습니다.

2. 데이터 완전 로컬 처리 (외부 전송 절대 없음)

본 프로그램이 수집한 모든 알림 텍스트와 캡처 이미지는 사용자가 직접 설정한 로컬 서버(또는 사용자의 개인 API 키가 연결된 LLM)로만 전송됩니다.

원작자(개발자)를 포함한 그 어떠한 제3자의 외부 서버로도 사용자의 개인 데이터가 몰래 수집되거나 전송되지 않습니다. * Project Lucia는 오픈소스 프로젝트입니다. 불안하신 분들은 언제든지 GitHub에 공개된 전체 소스 코드를 직접 확인하고 빌드하여 사용하실 수 있습니다.

3. 사용 시 주의 및 권고사항

개인 PC 사용 권장: 화면을 캡처하고 알림을 읽는 기능이 포함되어 있으므로, 공공장소의 PC나 사내 기밀 정보가 다뤄지는 업무용 PC에서의 사용은 권장하지 않습니다.

보안 정보 주의: 비밀번호, 금융 정보, 개인 민감 정보가 화면이나 알림에 노출되는 순간 AI가 이를 인지할 수 있으므로 각별한 주의가 필요합니다.

본 기능은 사용자의 편의를 위해 제공되며, 해당 기능 사용 시 발생하는 데이터 유출 등의 보안 사고에 대해서는 개발자가 책임을 지지 않습니다.

4. 기능 제어

화면 캡처 및 알림 감지 기능은 Lucia 설정 메뉴에서 언제든지 켜거나 끌 수 있습니다. 불안하시거나 불필요하다고 판단되시면 기능을 비활성화해 주세요.
